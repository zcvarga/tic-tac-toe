# Tic Tac Toe

Mini game I created while learning *React*. Basic two layer tic tac toe game.

## Installation
### Clone repository
`git clone https://github.com/zcvarga/tic-tac-toe `

`cd tic-tac-toe `

### Install dependecies
`npm install`

### Start app
`npm start`
